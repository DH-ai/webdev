The Website

Aim / Process

  • The idea behind this website is to showcase my skills.
  • Reason behind so many different pages where each one of them is looking is different form one another is what makes this website unique.
  • Being honest I have no idea about web development 10days ago. I have to learnt all of it and have to apply everything which I learn.
  • Now I'm pretty muchh confident that Now I can develop complex webpages or implement something cool on web.
  • I'm Learning I love to learn new things, and this website is a proof of that and to know more click.
  • I started with the home page i.e. 'index.html', for it I choose a matrix and dark theme representing a website related to computers and stuff.
  • Then I started working on my about page because I want to make a WordPress blog style feel keeping that in mind I explore different types of Web design, and that click me is something I feel to implement for experimentation.
  • Now for this website, I wanted to make it more interactive and with transitions and want to explore different styles so I came up with the idea of buttons to navigate to different projects.
  • I observed one thing starting from the first website to the end I feel that my skills developed really initially I was having problems even with the slide-in navigation boxes, but then I learnt about the IntersectionObserver and used it.
  • The rest of the pages and structures are then easy to do but had to brainstorm over the UI element of the site cause after all it is the thing that makes the site most fun to use and catchy.
